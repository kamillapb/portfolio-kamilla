# Portfólio Pessoal – Kamilla Philippsen Batista

Este projeto consiste no desenvolvimento de uma landing page funcional para apresentação do meu portfólio profissional, elaborada como atividade prática do curso de Introdução à Programação Front-End e Back-End.

A página foi construída utilizando HTML para estruturação do conteúdo, CSS para estilização visual e JavaScript para adicionar interatividade básica. O objetivo principal foi aplicar os conhecimentos adquiridos ao longo do curso, organizando informações profissionais em um único ambiente digital.

O portfólio apresenta meu nome completo, formação acadêmica, descrição profissional, habilidades técnicas e comportamentais, além de projetos desenvolvidos com indicação das tecnologias utilizadas.

Sou Bacharela em Sistemas de Informação e Pós-Graduada em Computação em Nuvem, atuando profissionalmente como Analista de Suporte em Infraestrutura. Atualmente, estou ampliando meus conhecimentos para desenvolvimento de software, com o objetivo de integrar infraestrutura, cloud e programação, evoluindo minha atuação na área de Computação em Nuvem.

Este projeto serve como base inicial para futuras atualizações, podendo ser expandido com novas funcionalidades, integração com back-end, hospedagem em nuvem e versionamento contínuo no GitHub.
